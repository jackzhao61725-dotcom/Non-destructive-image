# Scientific Baseline Layout

This directory is reserved for future numerical scientific baselines generated from the authoritative notebook in a complete scientific Python environment.

Current contents:

- `notebook_outputs.json` — lightweight stored-output baseline from Milestone 2.
- `atomic/` — planned `.npz` files for atomic-model and light-atom intermediate arrays.
- `imaging/` — planned `.npz` files for PCI, DGI, Faraday, camera, and shot-noise arrays.
- `multishot/` — planned `.npz` files for selected multi-shot evolution states.

Planned files are defined in `docs/baseline_specification.md`. Do not add fabricated numerical data. Baselines should be generated only by running the original notebook/reference implementation in a complete scientific Python environment.
