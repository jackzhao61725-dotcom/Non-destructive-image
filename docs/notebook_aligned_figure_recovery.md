# Notebook-Aligned Figure Recovery

## Purpose

This recovery starts from condensate generation because every later notebook
figure depends on the same upstream physical object: the Thomas-Fermi
condensate, its projected column density, and the coordinate grid used by the
imaging calculation.

The recovered figure is not a calibrated experimental prediction. It is a
notebook-aligned Version 1 recovery of one historical computational figure:
Stage 18.1, "constructing the condensate".

## What Stage 18.1 Shows

The figure has two panels.

Panel A shows 3D Thomas-Fermi density cuts through the trap centre:

```text
n_i(x) = n_peak * max(0, 1 - (x/R_i)^2)
```

The plotted units are:

- x-axis: position in micrometres;
- y-axis: density in `cm^-3`;
- three curves: `x (across)`, `y (along)`, and `z (across)`.

Panel B shows the projected column density along the x imaging axis:

```text
column_density_map = n_col[0] * max(0, 1 - y^2/R_y^2 - z^2/R_z^2)^1.5
```

The plotted units are:

- x-axis: `y (um)`;
- y-axis: `z (um)`;
- colourbar: `n_col (cm^-2)`.

The display follows the notebook choices:

- `Ngrid = 1024`;
- `FOV = 100 um`;
- `axis_demo = 0`;
- transverse plane: `y,z`;
- image extent: `[-50, 50, -50, 50] um`;
- column-density view limits: `xlim(-45, 45)`, `ylim(-12, 12)`;
- colour map: `viridis`;
- figure size: `11.8 x 4.6 in`.

## Remaining Uncertainty

The recovered figure reproduces the Stage 18.1 logic and display choices from
the notebook. It does not validate the physical model against experiment.

The notebook remains a historical Version 1 computational prototype. No
experimental absorption / RAI calibration has been applied. The recovered figure
therefore should be described as notebook-aligned and uncalibrated.
